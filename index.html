Some Features


Execute Rails commands visually

No more trying to remember basic rails commands! Use the inbuilt dialogs to execture rails commands. Many commands such as generate, destroy, scaffold and migrate are provided. The UI has been moulded to make it extremely easy for everyone to use - regardless of their experience!


In built terminal to execute commands

An in built terminal is available to execute all shell commands. The terminal can use any type of shell ( i.e zsh, bash). Furthermore, it can be used to run things like tmux and vim.


Database Visualisation

It is often quite hard to edit/see data in the terminal. Using this editor, the database can be seen and edited visually. The database functionality also offers features such as writing a SQL query, importing or exporting files, managing tables etc.


Generation of model and controller diagrams

Projects are often quite hard to visualise. This functionality generates model and controller diagrams of the chosen project. Using these diagrams, one can see the relations and even use them to make the archiecture of a system.


Cloning remote repositories

This functionality will allow cloning public and private repositories and then opening them in the editor. This will reduce the time and effort needed for anyone to get started on any project. The user also does not need to know how to use git!


P2P

When working in projects, one often needs to review the code written or discuss problems. To mitigate that issue, this editor provides a P2P functionality. This funcationlity allows video/audio and text based chat.
